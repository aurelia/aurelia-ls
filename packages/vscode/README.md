# Aurelia 2

Language tooling for Aurelia 2 templates.

## Features

- **Diagnostics** — Type errors in bindings, unknown elements and attributes
- **Completions** — Custom elements, attributes, bindables, expressions
- **Hover** — Type information and documentation
- **Go to Definition** — Navigate from template to component
- **Find References** — Locate usages across templates
- **Rename** — Rename symbols across TypeScript and HTML
- **Code Actions** — Quick fixes and refactorings

## Requirements

- Aurelia 2 project
- TypeScript or JavaScript

## Getting Started

Install the extension and open an Aurelia 2 project. The language server activates automatically when you open an HTML file.
