# App-Builder Fixtures

This folder is reserved for source generated by the app-builder algebra, not the
legacy authoring recipe layer.

`goldens/` contains materialized outputs for small app-builder selections so the
generated source can be inspected directly. Refresh them with:

```powershell
pnpm --filter @aurelia-ls/semantic-runtime fixtures:app-builder
```

Each golden includes the generated source files plus `app-builder.golden.json`,
which records the typed workflow selection, chosen pattern composition, Aurelia
lowering options, source policy, file preview, and expected semantic effect kinds.

Current goldens cover the minimal app shell plus a clean task-list starter. The
task-list golden is a domain-preset instantiation for inspecting add-item and
collection value-flow mechanics; the reusable pattern is the state-backed
collection list, not the task domain itself. Its records come from the explicit
`task-list.public-small` seed data set; seed data can be disabled or swapped
without changing the domain preset.
